# 📋 Attendance Management System

This is a web-based Attendance Management System developed to manage and track students and faculty attendance efficiently. The system allows users to register, log in, and maintain attendance records in a structured and organized manner.

The project is designed to reduce manual record-keeping and provide a digital solution for managing attendance data securely and accurately.

## Features

- Admin, Faculty and student registration and login system  
- Add and manage students and faculty records  
- Mark and update attendance  
- Organized attendance tracking  
- Clean and user-friendly interface  

## Technologies Used

- **Next.js (React framework)** with TypeScript  
- **TypeScript** for component logic  
- **Tailwind CSS** for styling UI  
- **JavaScript** for additional scripts  
- **HTML & CSS** for web pages  

## How to Run

Clone the repository, open the project folder in VS Code, configure the required backend or database if applicable, and run the main file or start the development server to access the system.
